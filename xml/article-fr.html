Flux de publication XML contrôlé qualité

Auteur : Gonzalo Zbinden · Date : 2025-10-15 · Langue : fr

Résumé

Cette démo illustre un flux éditorial où un XML sémantique est transformé en HTML accessible via XSLT et stylé avec CSS, prêt à être déployé sur GitHub Pages.

Introduction

La préparation, la structuration et la validation du XML avant publication exigent à la fois une sensibilité éditoriale et des compétences techniques solides.

Cet exemple reflète une séparation claire des responsabilités : XML pour le contenu, XSLT pour la présentation et CSS pour le style.

Fonctionnalités clés

  • Structure XML sémantique (titre, métadonnées, sections, références)
  • Transformation XSLT 1.0 vers un HTML5 conforme aux standards
  • Balises accessibles (titres, texte alternatif, zones de repérage)
  • Feuilles de style simples et portables avec thème sombre
